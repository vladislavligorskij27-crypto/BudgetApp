# 💎 Мои Финансы (My Finance App)

**Мои Финансы** — это современное и интуитивно понятное мобильное приложение для учета личных доходов и расходов. Проект разработан с акцентом на удобство использования, быстрый ввод данных и наглядную визуализацию текущего финансового состояния.

Приложение создано с использованием кроссплатформенного фреймворка **.NET MAUI** (C# / XAML) и предназначено для работы на устройствах под управлением ОС Android.

## 🚀 Основной функционал

* **Учет операций:** Быстрое добавление доходов и расходов с указанием точной суммы и названия.
* **Категоризация:** Поддержка различных категорий трат («Еда и продукты», «Транспорт», «Развлечения», «Квартплата и связь», «Одежда» и др.) для детальной аналитики.
* **Умная визуализация:** * Автоматический расчет текущего баланса, общих доходов и расходов.
    * Динамический индикатор (Progress Bar), наглядно отображающий процент потраченных средств относительно заработанных.
    * Адаптивные иконки в истории операций, меняющиеся в зависимости от выбранной категории.
* **Локальное хранение данных:** Вся история операций безопасно сохраняется в локальной памяти устройства (Preferences API). Данные не пропадают при закрытии или перезагрузке приложения.
* **Управление историей:** Возможность удаления отдельных записей или полной очистки истории в один клик.
* **Локализация:** Базовая валюта приложения — белорусский рубль (BYN). Защита от ошибок при вводе копеек (поддержка как точки, так и запятой).

## 🛠 Технологический стек

* **Язык программирования:** C#
* **Пользовательский интерфейс:** XAML
* **Фреймворк:** .NET 10 MAUI (Multi-platform App UI)
* **Архитектура:** Применение паттерна, близкого к MVVM (разделение бизнес-логики и UI)
* **Целевая платформа:** Android (API 21+)

## ⚙️ Установка и запуск (Для разработчиков)

Для запуска проекта на вашем локальном компьютере потребуется:

1.  Установленная среда разработки **Visual Studio 2022** (с установленной рабочей нагрузкой ".NET Multi-platform App UI development").
2.  Склонируйте данный репозиторий:
    ```bash
    git clone [https://github.com/ВАШ_ЛОГИН/BudgetApp.git](https://github.com/ВАШ_ЛОГИН/BudgetApp.git)
    ```
3.  Откройте файл решения `BudgetApp.sln` в Visual Studio.
4.  В качестве целевого устройства выберите эмулятор Android или подключенное физическое устройство.
5.  Нажмите `F5` для компиляции и запуска проекта.
    *Примечание: В настройках проекта (`BudgetApp.csproj`) отключена сборка для Windows, проект настроен строго под архитектуру Android (`android-arm;android-arm64`).*
